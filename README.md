# qBittorrent Mullvad
Run qBittorrent WebUI over Mullvad VPN Wireguard in Docker Compose 

## How
- copy sample.env to .env
- edit .env following setup guides
    - https://github.com/qbittorrent/docker-qbittorrent-nox
    - https://github.com/qdm12/gluetun-wiki/tree/main/setup#setup
- `make up` or `docker compose up`
- die
